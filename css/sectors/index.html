CSS Selectors

CSS selectors are patterns used to select and style HTML elements. They tell the browser “which element(s) should get the style?”

1. Basic Selectors

  • Tag Selector – targets HTML elements by their name.
    Example: h1 { color: red; } will style all <h1> tags.
    Use when you want to style all elements of the same type.

  • Class Selector – targets elements with a specific class.
    Example: .highlight { color: blue; } applies only to <p class="highlight">. Use when multiple elements need the same style.

  • ID Selector – targets a unique element with an ID.
    Example: #main { background: yellow; } applies only to <div id="main">. Use when one specific element needs a unique style.

2. Combinators

  • Descendant ( ) – targets elements inside another element.
    Example: div p = all <p> inside a <div>.

  • Child (>) – targets direct children only.
    Example: div > p = only the <p> that are direct children of <div>.

  • Adjacent Sibling (+) – targets the element immediately after another.
    Example: h2 + p = the first <p> right after an <h2>.

  • General Sibling (~) – targets all siblings that follow an element.
    Example: h2 ~ p = all <p> after an <h2> in the same parent.

3. Pseudo-Selectors

  • Pseudo-Classes – style elements based on state or position.
    Examples: a:hover { color: red; } → link changes color on hover. p:first-child { font-weight: bold; } → first paragraph inside parent is bold.

  • Pseudo-Elements – style parts of an element.
    Examples: p::first-letter { font-size: 24px; } → only the first letter of <p>. p::after { content: "✔"; } → adds text/icon after each paragraph.

4. Special Selectors

  • Attribute Selectors – target elements by attribute and value.
    Example: input[type="text"] { border: 1px solid black; }.

  • Universal Selector (*) – selects all elements on the page.
    Example: * { margin: 0; } resets margin for all.

  • Grouping Selector (,) – apply same style to multiple elements.
    Example: h1, h2, p { color: gray; }.

Tips

  • Use classes for reusable styles.
  • Use IDs only for unique elements.
  • Combinators help you target elements based on hierarchy.
  • Pseudo-classes are great for interactive effects.
  • Pseudo-elements let you style parts of content without changing HTML.
